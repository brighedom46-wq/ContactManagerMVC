# ContactManagerMVC

A simple ASP.NET Core MVC application for managing contacts.

## Features
- Add, edit, view, and delete contacts
- Category-based contact organization
- Server-side validation using Data Annotations
- Entity Framework Core with SQL Server LocalDB
- MVC architecture (Models, Views, Controllers)

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- C#

## How to Run
1. Clone the repository
2. Open `ContactManagerMVC.sln` in Visual Studio
3. Update the database if required
4. Run the application
